## 🎨 Design Philosophy

**"Digital Temple"** — Minimalist editorial meets sacred geometry

**Visual Pillars**:
- 🕉️ Sacred geometry (yantras, mandalas) - decorative elements
- ✍️ Editorial typography (Newsreader serif, large italics for headlines)
- ⬛ Sharp minimalism (zero border radius on ALL elements)
- 🔶 Copper/Gold palette (primary: #E05C00, secondary: #e9c349)
- 🌑 Dark surfaces (#0e0e0e to #353434) with warm accents
- 🔢 Space Mono for all numbers and timers

---

## 🎯 Key Design Rules

**NEVER compromise on these**:
1. **Border Radius**: ALWAYS 0px (sharp corners everywhere)
2. **Headlines**: ALWAYS use Newsreader serif (often italic)
3. **Numbers**: ALWAYS use Space Mono monospace
4. **Primary Color**: ALWAYS use copper (#E05C00) for main actions
5. **Background**: ALWAYS use deep blacks (#0e0e0e to #353434)

## 🎯 Tech Stack

- **Frontend**: Next.js 14 (App Router) + TypeScript
- **Styling**: Tailwind CSS (custom config with temple colors)
- **Backend**: Supabase (Auth + Database + Storage)
- **AI** (Phase 2+): FastAPI + OpenAI
- **Fonts**: Newsreader, Inter, Space Mono
- **Icons**: Material Symbols Outlined
