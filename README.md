# TAPASYA 
Tapasya (Tapa) — the disciplined skill mastery app for people who are serious about becoming great.

Tapasya is a **skill mastery tracker** that helps people build a disciplined daily practice around any skill — coding, music, gym, chess, writing, language learning — and track their journey toward **10,000 hours of mastery**.

The name comes from the Sanskrit word for intense, devoted self-discipline — the inner fire that forges a master. The short form **Tapa** is used conversationally, just like “Insta” for Instagram.

### The problem this app is solving : 
## The Problem

Most people want to master a skill. 
Almost none do. 
Not because they lack talent — but because **progress is invisible**, consistency breaks, and there is no system holding them accountable.

## How Tapasya Solves It

| The Problem | How Tapasya Solves It |
| --- | --- |
| Progress is invisible — hours feel wasted | Visual progress bar toward 10,000 hour goal |
| No motivation to show up daily | Streaks, badges, squad accountability |
| Distraction breaks every focus session | Built-in deep work timer with focus mode |
| No reflection or learning from practice | AI coach gives weekly insights from your data |
| Practicing alone feels lonely | Squads — small accountability groups of 3-5 |
| No structure to skill learning | Skill roadmaps — Level 1 to Level 5 per skill |
